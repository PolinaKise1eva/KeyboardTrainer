# Клавиатурный тренажёр с AI-генерацией

Десктопное приложение для тренировки слепой печати с элементами геймификации и нейросетевой генерацией текстов.  
Проект выполнен в рамках курсовой работы.

---

## Основные возможности

- **AI-генерация текстов и кода**  
  Встроенный модуль YandexGPT создаёт уникальные осмысленные тексты (RU/EN) и примеры кода (C, C++, Python, Java, Go и др.) нужной длины и сложности.
- **Локальная база текстов**  
  При отсутствии интернета автоматически используются готовые упражнения из SQLite (≈ 60 текстов на все комбинации языков/длин/сложностей).
- **Геймификация**  
  - Система опыта (XP) и уровней (1–10) с прогресс-баром.  
  - Звания: «Новичок», «Продолжающий», «Мастер».  
  - Цветная рамка аватара меняется при каждом новом уровне (10 цветов).  
  - Дневной минимум символов с возможностью настройки.
- **Тренировка набора кода**  
  Поддержка 8 языков программирования (C, C++, C#, Python, Java, Go, ASM, 1C). Три уровня сложности: лёгкий, средний, сложный.
- **Статистика**  
  Сохраняется история всех сессий (точность, скорость WPM, ошибки). Отображается в таблице по каждому языку и проблемный символ.
- **Загрузка своих файлов**  
  Возможность открыть `.txt` или файл с кодом и тренироваться на нём.
- **Современный интерфейс**  
  Тёмная тема, виртуальная клавиатура с подсветкой активной клавиши, автоматический перенос длинных строк, адаптивная прокрутка.
- **Автономная работа**  
  Все данные хранятся локально (SQLite). Для тренировки не требуется интернет. AI‑генерация опциональна.

## Технический стек

| Компонент | Технология |
|-----------|------------|
| Язык | C++17 |
| Графический интерфейс | Qt 6.5+ (Widgets, Network, Sql) |
| База данных | SQLite |
| AI-генерация | YandexGPT API (yandexgpt-lite) |
| Сборка | qmake или CMake 3.19+ |
| Среда разработки | Qt Creator |

## Запуск проекта (в Qt Creator)

1. **Склонируйте репозиторий**  
   ```bash
   git clone https://github.com/yourusername/KeyboardTrainer.git
   cd KeyboardTrainer
2. **Откройте проект в Qt Creator**
  Запустите Qt Creator.
  Нажмите Файл → Открыть файл или проект.
  Выберите файл проекта: CMakeLists.txt
3. **Настройте комплект (kit)**
Qt Creator автоматически предложит настроенный комплект для вашей системы (компилятор, версия Qt). Убедитесь, что выбрана версия Qt 6.5 или новее.
4. **Соберите и запустите**
Нажмите зелёную кнопку Run (или клавишу Ctrl+R).
Приложение соберётся и запустится прямо из среды.
